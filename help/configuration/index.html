Configuration

The Doodle can be configured in the service registration part. The options are as follows

  • Theme The global theme of the component.
  • DefaultStrokeSize The default brush stroke size
  • DefaultEraserSize The default size of the eraser brush
  • DefaultStrokeColor The default stroke color
  • BackgroundColor The background color of the Doodle
  • ColorPickerConfig The Color Picker Configuration
  • SizePickerConfig The Size Picker Configuration
  • CanvasConfig The Canvas Configuration
  • Html2CanvasConfig The default Html2Canvas Configuration
  • ToolbarConfig The Toolbar Configuration
  • ResizableElementConfig The Resizable Elements Configuration
  • BackgroundConfig The Backgrounds Configuration

ColorPickerConfig

  • FavouriteColors Gets or sets the favourite colors
  • ShowCustomColor Gets or sets a value indicating if the custom colors should be allowed

SizePickerConfig

  • FavouriteSizes Gets or sets the favourite sizes
  • ShowCustomNumber Gets or sets a value indicating if the custom selection should be available

CanvasConfig

  • GridSize Gets or sets the initial grid size
  • GridColor Gets or sets the initial Grid Color
  • GridType Gets or sets the initial Grid Type
  • UpdateResolution Gets or sets the update resolution for the canvas points
  • ResizableImages Gets or sets the available list of resizable images that can be used

ToolbarConfig

  • ShowToolbar Gets or sets a value indicating if the toolbar should be displayed
  • ShowCanvasGrid Gets or sets a value indicating if the canvas grid toolbar button should be displayed
  • ShowBackgroundPicker Gets or sets a value indicating if the background picker toolbar button should be displayed
  • ShowColorPicker Gets or sets a value indicating if the color picker toolbar button should be displayed
  • ShowSizePicker Gets or sets a value indicating if the stroke size picker toolbar button should be displayed
  • ShowDrawType Gets or sets a value indicating if the draw types should be displayed
  • ShowDrawTypeEraser Gets or sets a value indicating if the draw type erasor button should be displayed
  • ShowDrawTypePen Gets or sets a value indicating if the draw type pen button should be displayed
  • ShowDrawTypeLine Gets or sets a value indicating if the draw type line button should be displayed
  • ShowDrawTypeResizableText Gets or sets a value indicating if the draw type resizable text button should be displayed
  • ShowDrawTypeResizableImage Gets or sets a value indicating if the draw type resizable text button should be displayed
  • ShowDrawActions Gets or sets a value indicating if the draw actions should be displayed
  • ShowDrawActionsUndo Gets or sets a value indicating if the draw actions undo button should be displayed
  • ShowDrawActionsRedo Gets or sets a value indicating if the draw actions red button should be displayed
  • ShowDrawActionsClear Gets or sets a value indicating if the draw actions clear button should be displayed
  • ShowDrawActionsSave Gets or sets a value indicating if the draw actions save button should be displayed
  • ShowDrawActionsExport Gets or sets a value indicating if the draw actions export button should be displayed
  • ClearHistoryOnClear Gets or sets a value indicating if drawing history should be cleared on the clear button click

ResizableElementConfig

This configuration controls how the resizable text and image elements are displayed.

  • AllowMove Gets or sets a value indicating if the element can be moved
  • AllowResize Gets or sets a value indicating if the element can be resized
  • AutoHandleEvents Gets or sets a value indicating if the element should internally handle the select commands
  • Height Gets or sets the initial height of the element
  • Width Gets or sets the initial width of the element
  • Top Gets or sets the initial top of the element
  • Left Gets or sets the initial left of the element
  • xxx yyy
  • MinWidth Gets or sets the minimum width of the element
  • MinHeight Gets or sets the minimum height of the element

BackgroundConfig

  • BackgroundSources A list of sources that can be used as backgrounds on the Doodle
